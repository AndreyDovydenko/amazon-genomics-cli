--- title: About AGC linkTitle: About menu: main: weight: 40 --- {{< blocks/cover title="About AGC" image_anchor="bottom" height="min" >}} {{< /blocks/cover >}} {{% blocks/section %}} Amazon Genomics CLI, or AGC is an open-source command line tool developed by the AWS Health AI team to simplify running genomics workflows in the cloud. AGC takes care of deploying and managing the cloud infrastructure required to run workflows in common bioinformatics workflow languages such as WDL and Nextflow DSL allowing you to focus on analytics and workflow development. {{% /blocks/section %}}
